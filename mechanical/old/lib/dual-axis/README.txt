# Summary

FICBOX Dual-axis XY Joystick Module For Arduino Joystick Module Dummy

This is not a functional joystick, this is a dummy for 3d modeling enclosures or supports around a mini 3d joystick. This one is modeled after the FICBOX brand.

Importing this into your 3D modeling application allows you to see size and placement in the app without relying measurements alone.

Note, there are minor size differences between different brands, so please verify the model against the brand you are using before getting too far into the process in case measurements differ.
